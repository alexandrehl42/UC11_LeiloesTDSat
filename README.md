# Leilões TDSat
Sistema desenvolvido para auxiliar uma casa de leilões na organização de produtos, cadastro e listagem, utilizando interface gráfica em Java. O projeto faz parte das atividades acadêmicas relacionadas ao uso de versionamento com Git e GitHub.

# Sobre o projeto
O sistema permite:
- Cadastro de produtos
- Listagem de produtos registrados
- Conexão com banco de dados MySQL
- Interface gráfica construída com Java Swing (NetBeans)

O objetivo é demonstrar a estrutura inicial do sistema e preparar o ambiente para versionamento contínuo da aplicação.

# Tecnologias utilizadas
- Java (JDK + NetBeans) 
- MySQL
- MySQL Connector/J
- Git e GitHub para versionamento

# Estrutura geral
- src/ – Classes Java do sistema  
- build/ – Arquivos compilados gerados pelo NetBeans  
- lib/ – Conectores do MySQL utilizados para a conexão  
- nbproject/ – Configurações internas do NetBeans
